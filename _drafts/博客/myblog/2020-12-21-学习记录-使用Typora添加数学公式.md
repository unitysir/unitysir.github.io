## 行内与独行

### 行内公式

将公式插入到本行内

```latex
$公式内容$
\inline
```

例：坐标轴 ![img](使用Typora添加数学公式.assets/1410667-20190901135349117-159501400.png)

```latex
$xOy$
```

### 独行公式

公式单独占用一行

```latex
$$公式内容$$
\displaystyle
```

例：等差数列求和公式：

![img](使用Typora添加数学公式.assets/1410667-20190901135419581-1474916726.png)

```latex
$$S_n = \frac{1}{2}n(a_1 + a_n) = na_1 + \frac{n(n-1)}{2}d, n\in N^*$$
```
