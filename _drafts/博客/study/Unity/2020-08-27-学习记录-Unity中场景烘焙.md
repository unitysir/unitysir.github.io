# 场景基本烘焙

1. 将对象设置为 static 类型
2. 环境光设置为 color
3. 将 RealTime 改为 Baked
4. 将 intensity multiplier 改为 0
5. 取消勾选 实时光照
6. 勾选 全局烘焙，光照模型现选择 Subtractive
7. 将 间接分辨率设置为 1 ，将 光照贴图设置为 6，适当设置贴图距离和大小
8. 方向模型设置 Non-Directional
